# 兴趣班打卡

一个给孩子兴趣班使用的浏览器打卡记录工具，用来记录课程、课时消耗、作业反馈、请假、调课和补课情况。

在线使用地址：

https://sun9857.github.io/interest-class-checkin/

## 主要功能

- 课程管理：记录课程名称、上课星期、时间、总课时、剩余课时和每次消耗课时。
- 今日打卡：按今天的固定课程优先显示，也可以手动添加临时课或补课记录。
- 打卡记录：记录已上课、请假、调课、已补课、取消等状态。
- 课时统计：查看剩余课时、本月消耗、待补次数、已补次数和最近上课日期。
- 记录查询：按课程、状态和日期范围筛选历史记录。
- 备份导入导出：支持导出 JSON 备份、导出 CSV 记录、导入 JSON 备份。

## 怎么使用

### 1. 新增课程

1. 打开网页。
2. 进入「课程管理」。
3. 填写课程名称、上课星期、上课时间、总课时、剩余课时、每次消耗课时。
4. 点击「新增课程」。

### 2. 每天打卡

1. 进入「今日打卡」。
2. 选择课程、日期、时间、类型和状态。
3. 填写课时消耗、作业内容、老师反馈或备注。
4. 点击「保存打卡记录」。

状态说明：

- 已上课：会消耗课时。
- 请假：不会消耗课时，会计入待补。
- 调课：不会消耗课时，会计入待补。
- 已补课：会消耗课时。
- 取消：不会消耗课时。

### 3. 记录补课

如果某次补课是为了补之前的请假或调课：

1. 在「今日打卡」里新增记录。
2. 选择「关联请假/调课记录」。
3. 保存后，「课时统计」里的待补次数会减少，已补次数会增加。

### 4. 查询记录

1. 进入「记录查询」。
2. 可以按课程、状态、开始日期、结束日期筛选。
3. 点击「筛选记录」。

### 5. 查看统计

进入「课时统计」，可以查看每门课的：

- 剩余课时
- 本月消耗课时
- 待补次数
- 已补次数
- 最近上课日期

## 备份和恢复

数据默认保存在当前浏览器本地。为了避免换手机、换电脑、清理浏览器缓存后丢失数据，建议定期备份。

### 导出备份

1. 进入「设置备份」。
2. 点击「导出 JSON 备份」。
3. 保存下载的 `.json` 文件。

### 恢复备份

1. 进入「设置备份」。
2. 点击「导入 JSON 备份」。
3. 选择之前导出的 `.json` 文件。

注意：导入备份会覆盖当前浏览器里的数据。

### 导出表格记录

1. 进入「设置备份」。
2. 点击「导出 CSV 记录」。
3. 可以用 Excel 或 WPS 打开 CSV 文件查看历史记录。

## 数据和隐私说明

- 这个工具不需要注册账号。
- 数据保存在使用者自己的浏览器本地。
- 你看不到别人填写的课程和打卡数据。
- 别人也看不到你的本地数据。
- 如果换设备或换浏览器，需要先导出 JSON 备份，再在新设备导入。
